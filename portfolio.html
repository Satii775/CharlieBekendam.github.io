This project was with 3 others, its purpose was to utilize 360 video and have the user go inside a cessna and learn how to fly. I was the main 3D developer, video editor, and did some UI. It was fun to work in a group and be able to do 3D modeling and get more experience with Adobe Premiere Pro.

I practiced my coding techniques by making the code more modular to be used by multiple objects, and I practiced more with 3D modeling. It has been fun making this game because it is something that I have wanted for a long time, and I get to make my dream into an actual product.
